INICIO

DISEÑO DE SITIOS WEB - FASE 2

La siguiente investigación contiene información sobre, que significa ‘Model Box’ qué interpretación tiene en el desarrollo y en una página WEB, permitiendo entender la forma de utilizar adecuadamente los elementos que componen los estilos en una página, permitiendo dar una forma decorativa que permita al usuario no solo poder interactuar con la página, también crear un diseño agradable que el cliente logre percibir.

En el diseño de esta página se incluyen varios elementos que tal vez no son visibles, pero que al utilizar herramientas de depuración que tienen los navegadores hoy en día como el Google Chrome o el Internet Edge entre otros, permite visualizar los elementos de la página y cómo se organizaron, al igual también se juega un poco con etilos y una utilización del archivo css.

TEMA SELECCIONADO

MODEL BOX

Model Box: Hace referencia a la representación de un elemento en todos los navegadores dentro de un rectángulo o caja ‘Box’; según como se define en un documento HTML se visualizará como una caja rectangular.

En esta caja rectangular que funciona como un modelo estándar de cada elemento en css determina en su diseño el tamaño, su posición y sus propiedades

• Margen: margin

• Borde: border

• Relleno: padding

• Contenido: content

Margen / margin: es la parte que prácticamente separa el ‘Box Model’ de otros elementos y se define como la última área. Prácticamente delimita el límite de la caja.

Borde / border: Es una línea que rodea el borde de la caja y se utiliza para darle una apariencia al contorno de la caja.

Puede tener varios estilos según la traza de la línea que se desee como también puede darse un grosor. También se le puede definir un color tanto a la línea del borde como al fondo.

Relleno / padding: Es un área alrededor del contenido y es una separación que determinada entre el contenido y el borde, esto permite establecer la ubicación del contenido dentro de la caja de acuerdo al borde, según se defina entre sus medidas superior ‘top’, inferior ‘bottom’, izquierda ‘left’ y derecha ‘right’.

Contenido / content: se refiere al contenido del elemento donde esta el texto, las imágenes o cualquier otro elemento que se va a mostrar al usuario, es prácticamente el contenido real.
